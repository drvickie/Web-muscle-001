Nesting in HTML

Nesting in HTML is when one HTML element is placed inside another HTML element. This allows for multiple HTML tags to be applied to the same content. Nested elements should be placed two spaces further to the right of the element they are nested in. This spacing is called indentation and it is used to make HTML easier to read.
